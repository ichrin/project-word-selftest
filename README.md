# The Word selfTest Project 2023-2025
# The Tango selfTest Project 2025-2026
本项目依照伯克利软件分发协议（BSD License），最初在notabug上面开源。
本项目是一个简陋的单词自测程序。最初版本是在2023年的暑假写成的，用途是自测单词（废话）。
你也许注意到了最初版本是用来自测英语单词的，后来变成了日语单词。因为我上大学学日语去了。
你也许能自己把它改成一个自测任何语言的单词的程序。重制版本好好写了注释，里面的函数和变量名都是规范的（叉腰）。
自测的源文件（包含单词的文件）是逗号分隔文件。很直观，但是记得用UTF-8编码。
自测的源文件（包含单词的文件）必须和核心程序（也许叫core.py，也许叫test.py，取决于你取得的版本）在同一目录下。
为什么要把类似的句子说两遍。
为什么不把类似的句子合并在一起。

## 使用方法：
### 首先，安装Python（废话）。
如果你是macOS，请用homebrew或者macports安装新版本python，自带的python版本太老了。
如果你是Windows，请用微软商店安装python。
如果你是GNU/Linux发行版，请用自带的包管理器安装python，如果你用的是高贵的没有包管理器的Gentoo之流，请从源码编译python。btw, I use arch...
如果你是ChromeOS，请打开Debian Linux容器/虚拟机，然后用APT安装python。
如果你是FreeBSD/OpenBSD或者啥啥BSD，用包管理器安装python或从源码编译python。
如果你是HarmonyOS或者Android或者iOS/iPadOS，请自行去应用商店寻找python解释器，但请注意它们大概率无法使用图形界面，这边建议换个正常的桌面设备。

### 然后，运行.py文件（废话）。
一切顺利的话会出现一个图形界面。
一切顺利的话，一切就是顺利的。
